# CoResearcher Protocol

An open protocol for coordinating scientific work between humans and AI agents.

## Core Primitives

- **Question** - Strategic research direction
- **Action** - Executable scientific activity  
- **Review** - External validation
- **Provenance** - Complete traceability

## Goals

- Traceable scientific activity
- Human-agent collaboration
- Agent-agent review
- Open infrastructure
